# dotfiles

## Requirements

- rust: https://www.rust-lang.org/tools/install
- homebrew
  - macOS: https://brew.sh/
  - Linux: https://docs.brew.sh/Homebrew-on-Linux
- neovim: https://github.com/neovim/neovim/wiki/Installing-Neovim

## Install

```zsh
# pre-install requirements
$ ./install.sh
$ ./setup.sh
```
